# java-ex-array-left-shift

<h1> A simple java excersize to warm up</h1>

shift an array by the number of desired steps

Input: {1,2,3,4} , 1 <br />
Ouput: {2,3,4,1}  <br />
<br />
Input: {1,2,3,4}, 3 <br />
Output: {4,1,2,3} <br />
<br />
Input: {1,2,3,4}, 4 <br />
Output: {1,2,3,4} <br />
<br />
Input: {1,2,3,4}, 5 <br />
Output: java.lang.ArrayIndexOutOfBoundsException <br />
<br />
Input: {1,2,3,4}, 0 <br />
Output: {1,2,3,4} <br />
